### Algoritmo do exercício 1 ###

1- Criar uma estrutura de repetição;
2- Criar uma variável dentro da estrutura que vá servir como índice;
3- Estabelecer a regra da estrutura como: enquanto o índice for menor que o tamanho do Array, a repetição continuará a ocorrer;
4- Estabelecer a soma de 1 no valor da variável índice enquanto a regra da estrutura for verdadeira;
5- Estabelecer linha de código que será realizada enquanto a estrutura de repetição acontecer;
6- Linha de código deve ser composta por um comando que imprima o item do Array localizado na posição do índice;

### Algoritmo do exercício 2 ###

1- Estabelecer uma variável com o valor 0 que receberá a soma dos números contidos dentro do Array;
2- Criar uma estrutura de repetição;
3- Criar uma variável dentro da estrutura de repetição que recerá o valor do item atual do Array;
4- Estabelecer código que será realizado até o fim do looping da estrutura;
5- Código deve atualizar o valor da variável que irá conter a soma dos números presentes no Array sempre somando o valor atual dela ao valor da varíavel criada dentro do looping;
6- Deve ser impresso no console o valor da variável que guarda a soma dos números;

### Algoritmo do exercício 3 ###

1- Estabelecer uma variável com o valor 0 que receberá a soma dos números contidos dentro do Array;
2- Criar uma estrutura de repetição;
3- Criar uma variável dentro da estrutura de repetição que recerá o valor do item atual do Array;

5- Código deve atualizar o valor da variável que irá conter a soma dos números presentes no Array sempre somando o valor atual dela ao valor da varíavel criada dentro do looping;
6- Criar uma variável que conterá o quociente da variável da soma dos números do Array pelo número de itens contidos dentro do Array;
7- Imprimir o valor da variável anterior no console;
